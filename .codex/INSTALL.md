# ai-dev-kit for Codex

## Install

```bash
codex install github.com/BenWeekes/ai-dev-kit
```

## What it provides

- **Git conventions** — injected at session start (lowercase commits, no AI tool names, present tense)
- **Git skills** — ship (commit+push), pr (create PR), sync (rebase onto main)
- **Docs skills** — update (generate progressive disclosure docs), test (verify docs)

## Files

- `skills/ai-dev-kit/SKILL.md` — main conventions and skill directory
- `skills/ai-dev-kit/git/` — git workflow skills
- `skills/ai-dev-kit/docs/` — documentation skills
